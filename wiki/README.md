## 🎯 项目特色

- **双端支持**: 桌面端和移动端独立界面
- **现代技术栈**: Vue 3 + TypeScript + Vite + TDesign
- **类型安全**: 严格的 TypeScript 规范
- **组件化**: 模块化架构，易于维护
- **状态管理**: Pinia 状态管理
- **样式方案**: Tailwind CSS + Vant 移动组件

## 🚀 快速体验

```bash
# 克隆项目
git clone <repository-url>
cd ainote-web

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

访问 [http://localhost:5173](http://localhost:5173) 查看桌面端
访问 [http://localhost:5173/mobile](http://localhost:5173/mobile) 查看移动端
